# Summary: Task 2.1.2 Provider Configuration API

## Overview

Extended `JidoCode.Agents.LLMAgent` with runtime provider/model hot-swapping capabilities including validation and PubSub broadcasting.

## Changes Made

### Modified Files

1. **`lib/jido_code/agents/llm_agent.ex`**
   - `configure/2` - Reconfigures agent with new provider/model at runtime
   - `list_providers/0` - Returns available providers from ReqLLM registry
   - `list_models/1` - Returns model names for a given provider
   - `validate_provider/1` - Validates provider exists in registry
   - `validate_model/2` - Validates model exists for provider
   - `validate_api_key/1` - Validates API key is available for provider
   - `broadcast_config_change/2` - Broadcasts config change events to PubSub

2. **`test/jido_code/agents/llm_agent_test.exs`**
   - Added 8 new tests for configure/2, list_providers/0, list_models/1

### Key Implementation Details

- Uses `Jido.AI.Model.Registry.Adapter` APIs (jido_ai v0.5.3):
  - `list_providers/0` - Get available providers
  - `list_models/1` - Get models for provider
  - `model_exists?/2` - Check if model exists
- Uses `Jido.AI.Keyring.get/2` for API key validation
- Hot-swap stops existing AI agent and starts new one with new config
- Graceful fallback if registry unavailable
- Broadcasts `{:config_changed, old_config, new_config}` to PubSub

## API Usage

```elixir
# List available providers
{:ok, providers} = JidoCode.Agents.LLMAgent.list_providers()
# => {:ok, [:anthropic, :openai, :google, ...]}

# List models for a provider
{:ok, models} = JidoCode.Agents.LLMAgent.list_models(:anthropic)
# => {:ok, ["claude-3-5-sonnet-20241022", ...]}

# Reconfigure at runtime
:ok = JidoCode.Agents.LLMAgent.configure(pid, provider: :openai, model: "gpt-4o")

# Invalid provider returns error
{:error, "Provider 'invalid' not found..."} =
  JidoCode.Agents.LLMAgent.configure(pid, provider: :invalid)

# Invalid model returns error
{:error, "Model 'fake' not found for provider 'anthropic'..."} =
  JidoCode.Agents.LLMAgent.configure(pid, model: "fake")

# Missing API key returns error
{:error, "No API key found for provider 'openai'..."} =
  JidoCode.Agents.LLMAgent.configure(pid, provider: :openai, model: "gpt-4o")
```
